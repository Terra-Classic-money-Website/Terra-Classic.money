# Layer 2 Listing Policy

The main Terra-classic.money Ecosystem page is reserved for projects and resources that provide actual utility for LUNC holders.

Token-only, speculation-only, memecoin, or L2 projects that mainly provide a token or trading market are not listed on the main Ecosystem page.

Those projects belong on the separate Layer 2 website when eligible:

```text
https://l2.terra-classic.money
```

## Relationship To The Main Website

Main Ecosystem page:

- Terra Classic applications
- wallets
- bridges
- tools
- infrastructure
- network inspection resources
- governance and information resources
- projects with concrete utility for LUNC holders

Layer 2 website:

- L2 projects
- L2 tokens
- token ecosystems
- project-specific markets
- speculative or trading-focused projects that do not meet the main Ecosystem utility requirement

## Roadmap Boundary

Only projects listed on, or clearly eligible for, the main Ecosystem page may add roadmap entries to the main Terra-classic.money Roadmap page.

L2-only, token-only, or speculation-only projects should not use the main Roadmap page for visibility. Their roadmap content belongs on the Layer 2 website if that surface supports it.

## Reason For Separation

This separation protects the main website's neutrality and public-good role.

The main Terra-classic.money website should help users understand and use Terra Classic. It should not become a token-promotion or market-discovery board for projects whose only function is trading speculation.
