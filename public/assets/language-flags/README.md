# Language flag assets

Circular language flag SVGs downloaded from HatScripts circle-flags.

- Source: https://github.com/HatScripts/circle-flags
- Gallery: https://hatscripts.github.io/circle-flags/gallery.html
- License: MIT, copied in `LICENSE-circle-flags.md`

The Terra Classic website uses the language-specific SVGs from `flags/language/` except for `zh-CN`.
`zh-CN` maps to the main `flags/cn.svg` asset because the locale explicitly targets Simplified Chinese for China and the source set does not provide a separate `language/zh-cn.svg`.
